## 2. Calibrage : ÉCHELLE d'impression % (`c68e418`)
- `Calibrage` étendu `{dx, dy, ex, ey}` — ex/ey en % (pas 0,5, borné 90–110),
  migration auto des configs enregistrées, `styleCalibrage()` partagé
  (translate + scale origine haut-gauche) appliqué par les 5 documents.
- Steppers « Échelle d'impression (%) » Largeur/Hauteur ; miniature qui
  s'étire/rétrécit en direct. Compense les imprimantes qui étirent la page.
- Preuve : 101 %/98 % → aperçu CG `scale(1.01, 0.98)`.

## 3. DIMENSIONS du papier modifiables (`d361caa`)
Demande : pouvoir changer le format des fiches (ex. CG 10,5 × 21,2) sans code.
- `dimensions` par document dans printConfig (`DIMENSIONS_DEFAUT`,
  `getDimensionsDoc`, migration auto) ; les 5 documents tirent leur taille racine
  **ET** leur `@page` de la config.
- Bloc « Dimensions du papier (cm) » Largeur × Hauteur (pas 0,1, bornes 3–45)
  + bouton « ↺ format d'origine » ; en-tête et miniature suivent en direct.
- Preuve : hauteur CG → 25 cm → aperçu 105×250 mm + `@page 105mm 250mm`.
