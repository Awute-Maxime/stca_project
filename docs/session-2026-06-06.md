### Fix critique ELECTRON_RUN_AS_NODE
- **Problème** : `package.json` utilisait `cross-env ELECTRON_RUN_AS_NODE= electron-vite dev`. Electron vérifie l'EXISTENCE de la variable (pas sa valeur). Même `""` déclenchait le mode Node-only → `electron.app === undefined`.
- **Fix** : `"dev": "electron-vite dev"` (suppression de cross-env)
- **Résultat** : `npm run dev` fonctionne directement depuis le terminal utilisateur
